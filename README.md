Red social - Proyect

crear entorno (dentro de PROYECTO-ENTORNO/ENTORNO)
		python -m venv env1
activar entorno 
		cd env1/Scripts         actívate
		cd ..                 desactívate

instalar django
		pip install django

crear proyecto
		python manage.py startproject redsocial .

crear app
		python manage.py startapp redsocial
